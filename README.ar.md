# 🎞️ Archive Recover

تطبيق ويب مستضاف ذاتياً **لاستعادة فيديوهات يوتيوب المحذوفة من Internet Archive (Wayback Machine)** وتحويلها إلى مكتبة فيديو حديثة ونظيفة قابلة للتصفح — شاهد أي فيديو مستعاد مباشرة في متصفحك أو حمِّل الملف الأصلي.

مبني على أنبوب `yt-dlp` + Wayback CDX recovery pipeline.

---

## ✨ المميزات

- **استعادة بأي طريقة** — الصق قناة، `@handle`, اسم مستخدم قديم، رابط فيديو، أو معرف فيديو بـ 11 حرفاً. يكتشف كل مقطع مؤرشف يمكنه العثور عليه ويسحب أفضل جودة متاحة.
- **معرض فيديو حديث** — شبكة صور مصغرة متجاوبة مع ملصقات مولّدة تلقائياً، عناوين نظيفة، مدد، وأحجام ملفات.
- **شاهد في المتصفح** — مشغل مدمج مع دعم HTTP range (بدء فوري + تقديم) وتشغيل تلقائي للفيديو التالي.
- **بحث فوري** — رشّح المكتبة بأكملها بالعنوان أثناء الكتابة.
- **استعادة مباشرة** — ابدأ مهمة وشاهد التقدّم لحظة بلحظة (تم اكتشافه / تم تجربته / تم استعادته) عبر Server-Sent Events.
- **تحميل بنقرة واحدة** لأي ملف أصلي.

## 🚀 بدء سريع

```bash
npm install
PORT=3000 node server.js
# افتح http://localhost:3000
```

تُقرأ الفيديوهات المستعادة من `DATA_ROOT` (الافتراضي: `/home/khaled/recovered_videos`).
لتغييره:

```bash
DATA_ROOT=/path/to/videos PORT=3000 node server.js
```

### تشغيل إنتاجي (pm2 + HTTPS عام)

```bash
pm2 start ecosystem.config.cjs   # تطبيق الويب + نفق Cloudflare
pm2 save
```

الملف `ecosystem.config.cjs` يشغّل التطبيق **ونفق** Cloudflare سريع (`cloudflared`) للحصول على رابط عام `https://…trycloudflare.com`.

## 🧱 التقنيات

- **Node.js + Express** — API، دفق بالنطاق، تقدّم مباشر عبر SSE
- **yt-dlp** — محرك الاستعادة (روابط متعددة احتياطية، أفضل جودة)
- **ffmpeg / ffprobe** — توليد الصور المصغرة + حساب المدة (مخزّن على القرص)
- **واجهة أمامية بـ Vanilla JS** — بدون خطوة بناء، ملف `public/index.html` واحد

## 📦 واجهة API

| المسار | الغرض |
|-------|---------|
| `POST /api/recover` | بدء مهمة استعادة (`{ target, reuse }`) |
| `GET /api/files` | قائمة المكتبة (العنوان، المدة، الحجم، الامتداد) |
| `GET /api/stream` | تقدّم مهمة مباشر عبر SSE |
| `GET /thumb?file=` | صورة مصغرة JPEG مخزّنة |
| `GET /stream?file=` | دفق فيديو بدعم النطاق (للتشغيل في المتصفح) |
| `GET /download?file=` | تحميل الملف الأصلي |

## ⚙️ المتطلبات

يجب تثبيت `yt-dlp` و `ffmpeg` و `ffprobe` على الخادم.
